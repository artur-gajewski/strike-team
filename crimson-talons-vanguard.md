# ⚔️ Crimson Talons Vanguard
### Strike Team Character Sheets

> An elite rapid-response force deployed behind enemy lines. Five operants. One objective. No extraction plan.

---

## Game Rules Reference

| Stat | Description |
|------|-------------|
| **MOV** | Movement in inches per activation |
| **HP** | Hit Points — reduced by damage; operant is downed at 0 |
| **AC** | Armor Class — attacker must roll ≥ AC (d20 + attack bonus) to hit |
| **PER** | Perception modifier — added to d20 when spotting enemies or detecting threats |

**Weapon Columns:** Range (inches) · Attack Bonus (added to d20 to hit) · Damage Dice · Special
